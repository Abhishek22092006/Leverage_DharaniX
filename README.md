## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```id="z8r7dp"
git clone https://github.com/Abhishek22092006/Leverage_DharaniX.git
cd Leverage_DharaniX
```

---

## 🔧 Backend Setup

```id="9fx7bt"
cd backend
npm install
```

### Create `.env` file

```id="2a9b3f"
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

### Run backend

```id="l8v2ka"
npm start
```

---

## 🎨 Frontend Setup

```id="6gk2js"
cd ../frontend
npm install
npm run dev
```
